2016总结
发布于：2017-02-09 11:08:49

2016年来了，祝各位新年快乐，在2017年里身体健康、心想事成。

正在老家过春节。家的感觉真好，温暖舒服，心情畅快。很愉快地就决定把2016年的总结写了。

今年的总结加个新花样，就是在开篇给出一年中几个让我印象深刻的点：

  • 北京房价大幅上涨，学习了几个新词：暴涨、跳涨、报复性补涨、上车、站岗、踏空…
  • 早晚骑自行车上下班的路上，在非机动车道被后面的汽车按喇叭滴滴、多次“被”与汽车擦肩而过吓出一声冷汗、绿灯直行过马路时被右转弯的汽车别停
  • 汽车车牌摇号一直不中。因为政府规定而不能买车，算是属于我们这代人的时代印记吧
  • 手机app使用频次最高的是微信，其次是空气污染指数。年末冬天每个周末都重度雾霾，只能蜗居在家。入冬的时候新买了把吉他，周末被迫蜗居在家时就练练（其实我心里是想着没准以后靠这个挣钱呢）

当时不觉得，现在回顾起来，我的生活环境真的是“地狱模式”（空气恶劣、秩序混乱、房价高企…），好在我依然还在努力自娱自乐着。

BTW, 现代社会，我认为应该把衣食住行变成环食住行医教业：环境、食品、住房、交通、医疗、教育、就业。从国内范围来比较的话，北京在环境、住房、交通三方面都是负分。

变化

2016年我没什么大变化。只是从外貌上来看，变老了。（每当翻看前两年的照片，感觉尤其强烈)

媳妇儿变化很大。她今年开始正式带团队了，十来个人的团队。公司的管理培训、日常管理工作、更多地接触做领导和决策的人和事，让她的思考方式、看待人和事的方式都有了很大变化，水平和视野也有了很大提高。

更重要的职责意味着更全心的付出。今年媳妇儿比较辛苦，但上路之后会好起来的。Well done and keep it on!

工作

2015财年末尾的时候，我之前着手的一个项目从没有明确需求、自娱自乐的状态、到突然来了重要且紧急的需求状态（大概就这样这么简要描述一下）。结论就是评定2015财年年终绩效时，我没有什么给力的KPI产出（手动无奈悲剧脸）。

从中吸取到的教训就是：

  • 有问题、有疑问就及时扔出来，在日本学到的“報連相（ほうれんそう）”其实是广泛适用的
  • 根据老板的风格随机应变，目的是更快的沟通好双方的期望，目标一致的去努力
  • 一个人的能量有限，要学会找到杠杆、用杠杆来完成目标，来撬动更大的价值

2016年的工作，包括nodejs体系的web应用开发、数据接口服务开发、系统运维、命令行工具开发、内部一些npm模块开发。基本上就是这些了。

这一年在视野、思维层次、创意灵感上，并没有什么亮点。主要是积累了作为一个全栈工程师更多的实践经验，碰到了一些新的场景，比如分布式系统下一些功能的开发、消息和缓存思想的强化。所以算是积累的一年，希望2017年能有所爆发。

去年总结中我提到，2016年有一个目标，就是想办法带动团队中更多的人来分享他的积累。然而（以我的标准来看）并没有实现。

健康

大概从5月份开始，我下决心减肥了。（因为已经到了人生体重巅峰，再不减，到了冬天长膘的季节，画面太美不敢想象）

虽然在历史上，我曾若干次减肥成功，并且十分轻松。但现在回过头来看，这次减肥其实是一个全新的过程：

  • 学习了如何找到自己最舒服最高效的减肥方式
  • 学习理论上的减肥要点（严格饮食控制、进而养成良好的饮食习惯，原则就是消耗要大于摄入）

这次减肥，我是不知不觉地走上了十分科学健康的减肥道路。以前没有去了解这些东西，也能减下来。但现在看，那时候减肥主要靠节食饿肚子、并不科学，其实是有害身体的。而且以前我也不会主动去查找关于饮食、肥胖、脂肪、运动的知识。

可能是经过这几年，我已经有了更科学更主动的学习习惯了。这种意识到自己已经改变的时刻，很神奇。

最终效果很明显，从年初的78KG，到10月份的时候已经减到了63KG。总共15KG的减重。减掉脂肪的同时，肌肉也掉得很明显（哭）。能明显从胸部的厚度和手臂的粗细看出来。

三观

心态平和，对人友善。

一直以来我对人都是比较挑剔的。一方面是，如果碰到我觉得思维、道德、习惯在我的评判标准之下的人，就会表现出很强烈的厌恶感，避而远之。另一方面，我很看重人的独立自主性，对于我认为他能自己搞定的事情，我不会也不愿插手。

这个问题困扰了我多年，也给我带来过一次又一次的精神和物质上的伤害。待人友善，能获得比待人严苛更实实在在的好处，这是我很明白的。去年一年时间我都在努力解决它，目前效果还比较满意。以后继续努力。

解决方式的核心就是努力锻炼自己的“同理心”：尊重每个人的存在 -> 走进他的场景和思维 -> 按双方价值最大化的方式行动。最后发现这么做的花费也没那么大，所以一直进行这样的循环就好了。

PS: 我不是要做一个“好好先生”，而是做一个“友善的人”。

    • 一个人的朝圣，富兰克林传，跟任何人都聊得来，谈话的力量，汽车是怎样跑起来的，岛上书店，把时间当朋友，生活中的经济学，人生的智慧（60%），李鸿章与晚清四十年
